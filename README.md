# `%slab` #

A tool for launching and maintaining portal digital organizations (PDOs).

## Install ##

```bash
nvm install 22
nvm use 22
npm install
npx prisma db push
npm run seed
```

## Build/Develop ##

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see
the result.
